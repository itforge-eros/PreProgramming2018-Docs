# Understanding Directory

### **C**hange **D**irectory

```
cd <path>
```

### **L**ist**s**

```
ls
```

Modifiers:
- L Long List
- A All
- S Sorted (by child file)

### **P**rint **W**orking **D**irectory

```
pwd
```
